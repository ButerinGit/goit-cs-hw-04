# goit-cs-hw-04

Домашнє завдання: **Конкурентність та паралелізм**

Програма виконує паралельний пошук заданих ключових слів у текстових файлах двома способами:
- за допомогою `threading` (багатопотоковість),
- за допомогою `multiprocessing` (багатопроцесорність).

## Структура проєкту

- `data/` — вхідні `.txt` файли для пошуку.
- `common.py` — спільні функції для роботи з файлами.
- `threading_version.py` — реалізація паралельного пошуку з використанням потоків.
- `multiprocessing_version.py` — реалізація паралельного пошуку з використанням процесів.
- `main.py` — точка входу: запуск обох версій, вимірювання часу, порівняння результатів.
- `README.md` — опис проєкту.

## Логіка роботи

- Програма збирає всі `.txt` файли з папки `data/`.
- Для кожного ключового слова повертається список файлів, де це слово зустрічається.
- Обидві версії повертають словник:
  ```python
  {
      "слово": ["data/test1.txt", "data/test2.txt", ...],
      ...
  }
